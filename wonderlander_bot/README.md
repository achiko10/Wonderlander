# Wonderlander Wellness Telegram Bot

ეს არის Telegram ბოტი, რომელიც დაწერილია `aiogram` ბიბლიოთეკის გამოყენებით.

## როგორ გავუშვათ ლოკალურად (VS Code-ში)

1. **დააინსტალირეთ Python**: დარწმუნდით, რომ თქვენს კომპიუტერში დაინსტალირებულია Python 3.8 ან უფრო ახალი ვერსია.
2. **დააინსტალირეთ საჭირო ბიბლიოთეკა**:
   გახსენით ტერმინალი VS Code-ში და გაუშვით შემდეგი ბრძანება:
   ```bash
   pip install aiogram
   ```
3. **ფაილების სტრუქტურა**:
   შექმენით საქაღალდე და შიგნით განათავსეთ შემდეგი 3 ფაილი:
   - `config.py` (ბოტის ტოკენისთვის)
   - `keyboards.py` (ღილაკებისთვის)
   - `bot.py` (მთავარი ლოგიკისთვის)
4. **ბოტის გაშვება**:
   ტერმინალში გაუშვით:
   ```bash
   python bot.py
   ```
